# neoride Service Website

This repository contains the source code for the neoride service website. The site provides clear information about the platform, its features, and the policies that govern its use. It includes:

About the Service – What neoride offers, how the platform works, and key benefits for users.

About Us – Company background, mission, and the team behind neoride.

Privacy Policy – Details on data handling, storage, and user privacy practices.

Terms & Conditions – Rules, responsibilities, and legal terms for using neoride.

Support & Contact – Channels for customer support, queries, and feedback.

The repository is organized for easy maintenance and updates as the platform evolves.
